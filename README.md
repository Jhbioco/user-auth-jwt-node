# user-auth-jwt-node
Nodejs - user authentication with json web token and password hashing with bcrypt.

/*This is a simple implementation of user authentication using json web token, and bcrypt to hashing passwords.*/

## Dependencies
```
npm i express
npm i Joi
npm i mongoose
npm i bcrypt 
npm i jsonwebtoken
npm i config

```
